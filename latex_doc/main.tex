\section{Standard RPS}


The standard (general) rock paper scissors (RPS) payoff matrix is defined as:

\begin{align}
\begin{bmatrix}
    a & c & b\\
    b & a & c\\
    c & b & a
\end{bmatrix}
\end{align}

Let \( x \), \( y \), and \( z \) represent the fraction of players using rock, paper, and scissors respectively. Since \( z \) can be eliminated, we define:
\[
z = 1 - x - y
\]


Expected payoffs for each strategy can then be defined as follows:
\begin{align}
   \pi_R = a x + b \left(- x - y + 1\right) + c y \\
   \pi_P = a y + b x + c \left(- x - y + 1\right) \\
   \pi_S = a \left(- x - y + 1\right) + b y + c x
\end{align}


The replicator equations can be defined for $(x, y)$ using the standard formula:
\begin{equation}
    \dot x_i = x_i(\pi_i(x) - \langle \pi(x) \rangle)
\end{equation}

where the average payoff (or mean fitness) in the population is:
\[
\langle \pi(x) \rangle = x \pi_R + y \pi_P + z \pi_S
\]

This leads to the following replicator equations for x and y:

\begin{equation}
    \begin{split}
        \dot{x} &= x y \left(a x - a y - b x + b \left(-x - y + 1\right) + c y 
        - c \left(-x - y + 1\right)\right) \\
        & + x \left(-x - y + 1\right) \left(a x - a \left(-x - y + 1\right)
        - b y + b \left(-x - y + 1\right) - c x + c y\right)
    \end{split}
\end{equation}

\begin{equation}
    \begin{split}
        \dot{y} &= x y \left(- a x + a y + b x - b \left(- x - y + 1\right) - c y 
        + c \left(- x - y + 1\right)\right) \\
        & + y \left(- x - y + 1\right) \left(a y - a \left(- x - y + 1\right)
        + b x - b y - c x + c \left(- x - y + 1\right)\right)
    \end{split}
\end{equation}


Defining $F = \dot x, G = \dot y$, the following derivatives form the Jacobian used to determine the stability of the system:
\begin{align}
    \frac{dF}{dx} &= x y (a - 2b + c) 
    + x (2a - b - c)(-x - y + 1) \nonumber \\
    &\quad - x \left(a x - a(-x - y + 1) - b y + b(-x - y + 1) - c x + c y\right) \nonumber \\
    &\quad + y \left(a x - a y - b x + b(-x - y + 1) + c y - c(-x - y + 1)\right) \nonumber \\
    &\quad + (-x - y + 1) \left(a x - a(-x - y + 1) - b y + b(-x - y + 1) - c x + c y\right) \\
    \frac{dF}{dy} &= x y (-a - b + 2c) 
    + x (a - 2b + c)(-x - y + 1) \nonumber \\
    &\quad + x \left(a x - a y - b x + b(-x - y + 1) + c y - c(-x - y + 1)\right) \nonumber \\
    &\quad - x \left(a x - a(-x - y + 1) - b y + b(-x - y + 1) - c x + c y\right) \\
    \frac{dG}{dx} &= x y (-a + 2b - c)
    + y (a + b - 2c)(-x -y + 1) \nonumber \\
    &\quad +y (-ax + ay + bx - b (-x -y + 1) - cy + c (-x-y+1)) \nonumber \\
    &\quad -y (ay - a(-x-y+1) + bx - by - cx + c(-x-y+1)) \\
    \frac{dG}{dy} &= x y (a + b - 2c)
    + x \left( -a x + a y + b x - b(-x - y + 1) - c y + c(-x - y + 1) \right) \nonumber \\
    &\quad + y (2a - b - c)(-x - y + 1) \nonumber \\
    &\quad - y \left( a y - a(-x - y + 1) + b x - b y - c x + c(-x - y + 1) \right) \nonumber \\
    &\quad + (-x - y + 1) \left( a y - a(-x - y + 1) + b x - b y - c x + c(-x - y + 1) \right)
\end{align}


The Jacobian is as follows:
\[
J = 
\begin{bmatrix}
\frac{\partial F}{\partial x} & \frac{\partial F}{\partial y} \\
\frac{\partial G}{\partial x} & \frac{\partial G}{\partial y}
\end{bmatrix}
\]


Eigenvalues at the standard RPS game (a = 0, b = 1, c = -1):
\[
\left\{
    -\frac{1}{2} \sqrt{16 x^{2} + 16 x y - 16 x + 16 y^{2} - 16 y + 4} \,, \quad
    \frac{1}{2} \sqrt{16 x^{2} + 16 x y - 16 x + 16 y^{2} - 16 y + 4}
\right\}
\]

At the internal fixed point of $(x,y) = (\frac{1}{3}, \frac{1}{3})$, 
$\lambda_{1,2} =\left\{- \frac{\sqrt{3}}{3} i, \frac{\sqrt{3}}{3} i\right\}$

\par
No real parts of any eigenvalues, and imaginary components exist therefore neutral stability in standard RPS.
These eigenvalues are the solution at the internal fixed point of 
the standard RPS game. They tell us that the game follows neutral oscillations
about the internal fixed point (in the deterministic dynamics) as they contain no real parts (poincare diagram).


When $c = -0.8$
\begin{equation}
\lambda_{1}, \lambda_2 = \left\{-\frac{1}{30} + \frac{3 \sqrt{3}}{10}i,
-\frac{1}{30} - \frac{3 \sqrt{3}}{10}i\right\}
\end{equation}
Since the real parts of the eigenvalues are negative, 
$\operatorname{Re}(\lambda_{1,2}) = -\frac{1}{30} < 0$, the fixed point $\left(\frac{1}{3}, \frac{1}{3}\right)$
is linearly stable, so the populations will converge towards the central fixed point over time.
The imaginary component indicates the oscillatory dynamics so the population will spiral inwards 
towards the center. When $c = -0.8$, the populations converge towards a stable
coexistence. For this particular case of RPS a drift reversal can occur in finite populations.


\section{Combined game derivations}


Combined game payoff matrix:

\begin{align}
\label{AugRpsMatrix}
\begin{bmatrix}
    a & c & b & \gamma\\
    b & a & c & \gamma\\
    c & b & a & \gamma\\
    a + \beta & a + \beta & a + \beta & 0
\end{bmatrix}
\end{align}

\begin{align} \nonumber
\label{CompositeGame}
\begin{bmatrix}
    \langle RPS \rangle & \gamma\\
    \beta & 0 \\
\end{bmatrix}
\end{align}

Where the strategies are labelled as R,P,S and +.
From this the Jacobian matrix can be derived with dimensions $3 \times 3$ after using the trick to remove one replicator equation. (q = 1 - x - y - z)

\subsection{Payoffs}
Firstly the payoffs are defined for the augmented game:
\begin{align}
    \pi_R &= a x + b z + c y + \gamma (- x - y - z + 1)\\
    \pi_P &= a y + b x + c z + \gamma (- x - y - z +1) \\
    \pi_S &= a z + b y + c x + \gamma (- x - y - z + 1)\\
    \pi_+ &= x (a + \beta) + y (a + \beta) + z (a + \beta)
\end{align}

\subsection{Replicator equations}

From these payoffs, the standard payoff difference replicator equations can be defined, 
note again we need only 3 for the 4 strategy game as the fourth strategy can be 
elimated using the same trick as RPS, reducing the complexity slightly.

\begin{align} 
    \dot x &= x (a x + b z + c y + \gamma (- x - y - z + 1) - x (a x + b z + c y + \gamma (- x - y - z + 1)) \nonumber \\
    &- y (a y + b x + c z + \gamma (- x - y - z + 1)) - z (a z + b y + c x + \gamma (- x - y - z + 1)) \nonumber \\
    &- (x (a + \beta) + y (a + \beta) + z (a + \beta)) (- x - y - z + 1))
\end{align}

\begin{align}
\dot{y} &= y \big(a y + b x + c z + \gamma (-x - y - z + 1)
- x (a x + b z + c y + \gamma (-x - y - z + 1)) \nonumber \\
&\quad - y (a y + b x + c z + \gamma (-x - y - z + 1))
- z (a z + b y + c x + \gamma (-x - y - z + 1)) \nonumber \\
&\quad - (x (a + \beta) + y (a + \beta) + z (a + \beta)) (-x - y - z + 1) \big)
\end{align}


\begin{align}
\dot{z} &= z \big(a z + b y + c x + \gamma (-x - y - z + 1)
- x (a x + b z + c y + \gamma (-x - y - z + 1)) \nonumber \\
&\quad - y (a y + b x + c z + \gamma (-x - y - z + 1))
- z (a z + b y + c x + \gamma (-x - y - z + 1)) \nonumber \\
&\quad - (x (a + \beta) + y (a + \beta) + z (a + \beta)) (-x - y - z + 1) \big)
\end{align}

\noindent
Reduced the dimensions to just 3 replicator equations in x, y and z as q = 1 - x - y - z. \\
Jacobian formed as before, $F = \dot{x}, G = \dot{y}, P = \dot{z}$
\[
J = 
\begin{bmatrix}
\frac{\partial F}{\partial x} & \frac{\partial F}{\partial y} &\frac{\partial F}{\partial z} \\[6pt]
\frac{\partial G}{\partial x} & \frac{\partial G}{\partial y} &\frac{\partial G}{\partial z} \\[6pt]
\frac{\partial P}{\partial x} & \frac{\partial P}{\partial y} & \frac{\partial P}{\partial z}
\end{bmatrix}
\]


Once again the eigenvalues of this Jacobian matrix can be determined and tell us about the 
stability of the system of replicator equations derived for the combined game.
I have computed these values numerically for a particular instance of the game below.
The symbolic values for the eigenvalues are very large and bulky so are omitted. They can be generated using the 
$aug\_rps.py$ file included in the repository.

\section{Interaction processes \ simulations}

To analyze the effects of games in finite populations it is necessary to define
microscopic update processes for how the agents in the populations interact 
and also compute the probabilities for an agent of one strategy to switch to another when
they come into contact in the simulation. 

In order to achieve this I focus on 3 different interaction processes previously studied, 
inspired from biology namely the Moran process, the Local update process and the Fermi process.



\subsection{Transition probabilities}

For the dynamics in a finite population, let $i$, $j$, $k$, and $N-i-j-k$, be the 
be the discrete counts of members of each strategy respectively.


\subsubsection{Moran}

In the Moran process at each time step an individual is chosen for reproduction
proportional to their fitness within the population. Interestingly 
this process requires that each agent has access to some global average population fitness
$\langle \pi \rangle$ since they are selected proportional to the average population fitness 
and not individual interactions with a single agent. An agent is also chosen uniformly at random 
to be killed and replaced by the agent selected for reproduction.


\begin{figure}[h]
    \centering
    \includegraphics[width=0.7\linewidth]{images/moran_time_series.png}
    \caption{population size 60000 Moran process, along with numerical solution to adjusted dynamics.}
    \label{fig:placeholder}
\end{figure}

For the Moran process, for a single
transition ($R \rightarrow +$) the transition probabilities are as follows:
\begin{equation}
    T^{R \rightarrow +} = \frac{1}{2} \frac{1 - w + w \pi_+}{1 - w + w \langle \pi \rangle} \frac{i}{N} \frac{N - i - j - k}{N}
\end{equation}  
\begin{equation}
    T^{+ \rightarrow R} = \frac{1}{2} \frac{1 - w + w \pi_R}{1 - w + w \langle \pi \rangle} \frac{i}{N} \frac{N - i - j - k}{N}
\end{equation}  

Also transitions within RPS

\begin{equation}
    T^{R \rightarrow S} = \frac{1}{2} \frac{1 - w + w \pi_S}{1 - w + w \langle \pi \rangle} \frac{i}{N} \frac{k}{N}
\end{equation}  

\begin{equation}
    T^{S \rightarrow R} = \frac{1}{2} \frac{1 - w + w \pi_R}{1 - w + w \langle \pi \rangle} \frac{i}{N} \frac{k}{N}
\end{equation}  

\par
The rest of the transitions, $T^{R \rightarrow P}, T^{P \rightarrow R}, T^{S \rightarrow P}, T^{P \rightarrow S}, T^{P \rightarrow +}, T^{+ \rightarrow P}, T^{ S \rightarrow +}, T^{+ \rightarrow S}$
formed from permuations ($R, P, S, +$) and ($i, j, k, N-i-j-k$) \\


The parameter $w$ represents the selection pressure which 
determines how much the transition probabilities depend on 
the fitness differences between individuals. Weak selection
increases the impact of stochastic drift in the dynamics
and is often the biologically relevant case.

\subsubsection{Local update}

For the Local update process at each time step 2 randomly
selected individuals $a,b$ are chosen for reproduction.
Individual $b$ takes individual $a$'s strategy proportional
to the difference between their individual payoffs. Note that
there is not global average payoff here, so this could be 
a more realistic assumption in nature where individuals would 
not have this knowledge.

Individual $b$ switches to individual $b$'s strategy with the probability:
\begin{equation}
p = \frac{1}{2} +  \frac{w}{2} \frac{\pi_a - \pi_b}{\Delta \pi_{max}}
\end{equation}
Where $\Delta \pi_{max}$ is the maximum possible difference in payoff
between two individuals.

It follows that the transition probabilities for the 
populations in the combined game can be defined as:


\begin{equation}
  T^{R \rightarrow +} = \left (\frac{1}{2} + \frac{w}{2} \frac{\pi_+ - \pi_R}{\Delta \pi_{\text{max}}} \right ) \frac{i}{N} \frac{N - i - j - k}{N}
\end{equation}
for the case of $R \rightarrow +$, and the rest can again be 
written as permutations of $R,P,S,+$, and $i,j,k,N-i-j-k$.

    \centering

    \caption{Time series of local update with population 20000, simulation trajectory normalized, compared with the numerical solution of the replicator equations derived from FP equation.}
    \label{fig:placeholder}
\end{figure}

\subsubsection{Fermi process}

Similarly to the local update process two individuals are selected
at each time step and individual $b$ taks individual $a$'s strategy
with the probability:
\begin{equation}
p = \frac{1}{1 + e^{-w(\pi_a - \pi_b)}}
\end{equation}

The transition probabilities:

\begin{equation}
T^{R \rightarrow +} = \left( \frac{1}{1 + e^{-w(\pi_a - \pi_b)}}\right) \frac{i}{N} \frac{N-i-j-k}{N}
\end{equation}


\subsubsection{General form}
Generally the transition probabilites for any of the 
interaction processes ($R \rightarrow +$) can be written as:
\begin{equation}
  \phi(b \rightarrow a) \frac{N_a N_b}{N^2}
\end{equation}
Given an appropriate $\phi$ reproductive function which are defined for each process.

\begin{align}
\phi_{MO}(\pi_a, \langle \pi \rangle, w) &= \frac{1}{2} \frac{1 - w + w \pi_a}{1 - w + w \langle \pi \rangle} \\
\phi_{LU}(\pi_a, \pi_b, w) &= \frac{1}{2} +  \frac{w}{2} \frac{\pi_a - \pi_b}{\Delta \pi_{max}} \\
\phi_{FP}(\pi_a, \pi_b, w) &= \frac{1}{1 + e^{-w(\pi_a - \pi_b)}}
\end{align}

\begin{verbatim}
# Returns the transition probabilities given a repoductive function and payoffs.
# Returns dict indexed by T_RS for example between Rock and Scissors
# The 4th strategy is labelled as L
def transition_probs(reproductive_func, payoffs : list):
  # x y z, q = 1 - x - y - z

  names = ["R", "P", "S", "L"]
  freqs = [x, y, z, q]

  return {
      f"T_{a}{b}": reproductive_func(payoffs, i, j, w_sym) * freqs[i] * freqs[j]
      for i, a in enumerate(names)
      for j, b in enumerate(names)
      if i != j
  }
\end{verbatim}
I have implemented this method in $aug\_rps.py$, which can
compute these transition probabilites symbolically for 
both derivation of larger equations and also for numerical 
method solutions later on. It takes a reproduction function $\phi$
for a list of payoffs for each strategy.
The repoductive functions have to be defined in a general way,
so take the payoffs of all strategies, then the index $i,j$
which determines the strategies of individuals $a,b$ described above
and $w$.

\begin{verbatim}
def local_reproductive_func(payoffs, i, j, w):
  diff = payoffs[j] - payoffs[i]
  return (1/2) + (w / 2) * (diff / delta_pi)

def moran_reproductive_func(payoffs, i, j, w):
  payoff = payoffs[j]
  numerator = 1 - w + w * payoff
  denominator = 1 - w + w * averagePayoff
  return numerator / denominator

def fermi_reproductive_func(payoffs, i, j, w):
  diff = payoffs[j] - payoffs[i]
  return 1 / (1 + sp.exp((-w) * (diff)))

\end{verbatim}
These are easily defined from the formal equations, so more can 
be added with new simple methods.
Average payoff accessible globally for use in other formulas.
The methods in this file are used for calculating
exact values for the derived formula to complement the 
simulated values and also to produce exact deterministic
trajectories in the infinite population size case very 
efficiently as running the simulations for very large $N$ is 
very computationally expensive.
These transitions can be used to numerically solve the 
adjusted replicator dynamics for each of these processes and used
to plot deterministic trajectories (see more when plotting 
trajectories below.)
